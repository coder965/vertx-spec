# TB0001 - URI分类表

| 路径前缀 | 备注 | 认证模式 |
| :--- | :--- | :--- |
| /v1/\* | 纯开放API模式，其中v1为当前平台的版本号。 | 开放模式 |
| /v1/api/\* | 平台内部前端访问的基础模式，内部App或平台本身调用 | 平台模式（需认证） |
| /v1/open/\* | 由平台发布的开放性接口专用Api | 三方模式（需认证） |
| /v1/tp/{name}/\* | 开放给合作伙伴的接口专用文档 | 集成模式（需认证） |
| /v1/app/{name}/\* | 平台运行的App独立开发认证模式 | 终端模式（需认证） |
